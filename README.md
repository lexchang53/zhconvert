# 🔀 繁簡及兩岸地區用語轉換工具 (zhconvert)

這是一個專為 AI Agent 設計的隨插即用（Plug-and-Play）技能。本工具能透過調用「繁化姬」API，批次進行 **Markdown、純文字 (TXT)、Word (.docx)、Excel (.xlsx) 和 PowerPoint (.pptx)** 檔案的繁簡字元與兩岸習慣用語在地化轉換。

本工具最核心的技術優勢在於：**在進行簡繁用語轉換的同時，能 100% 完美保留 Office 檔案原本的樣式、字型格式與公式計算，且支援檔案名稱與內容的同步簡繁轉換。**

---

## ✨ 核心特色

1. **五種轉換模式 (Modes)**：
   * **`Taiwan` (台灣化，預設)**：簡轉繁，並轉換為台灣在地習慣用語（例如將 `合同` ➔ `契約`；`信息` ➔ `資訊`；`軟件` ➔ `軟體`；`支持` ➔ `支援`）。
   * **`Traditional` (純字元簡轉繁)**：僅進行簡繁字元轉換，保留原本的詞彙習慣（不作用語替換）。
   * **`Hongkong` (香港化)**：簡/繁轉為香港繁體，並進行香港在地習慣用語轉換。
   * **`Simplified` (純字元繁轉簡)**：純繁體轉為簡體，保留原本的繁體詞彙習慣。
   * **`China` (中國化)**：繁轉簡，並轉換為大陸在地習慣用語。
2. **完美格式保留 (Format Preservation)**：
   * **Word (.docx)**：逐個 Run 進行文字替換，字型、大小、色彩與表格排版完全不會跑版。
   * **Excel (.xlsx)**：自動跳過以 `=` 開頭的公式儲存格，保護函數公式不被破壞，同時保留儲存格樣式。
   * **PowerPoint (.pptx)**：完整替換文字框與投影片表格中的文字，保留原版面設計與動畫。
   * **Markdown / TXT**：保留 Markdown 語法格式不受轉換干擾。
3. **安全防護（備份舊檔）**：
   * 預設會自動將原始簡體檔案重新命名備份為 `[原檔名]_old.[副檔名]`，轉換後的結果則寫入原檔名中（亦支援檔案名稱同步繁簡轉換），確保文件內部的交叉連結不失效。
   * 只有在命令中明確加上 `--overwrite` 時，才會直接覆蓋而不備份。
4. **批次資料夾處理**：
   * 支援遞迴掃描整個資料夾下所有支援的文件，並自動排除 `*_old` 備份檔以防重複處理。

## 🛠️ 環境與網路需求

* **有網路連線環境**：本技能必須連線至繁化姬 API (api.zhconvert.org)，離線狀態下無法執行。
* **Python 依賴套件**：`requests`、`python-docx`、`openpyxl`、`python-pptx`。

> 💡 **按需載入與免手動安裝設計**：
> - 本工具採用 **「按需延遲導入 (Lazy Import)」** 機制。如果您僅轉換 Markdown 或 TXT 純文字檔案，執行時將**完全不會**檢查或在背景下載 Office 依賴套件，極致輕量。
> - 若處理的是 Office 檔案且環境缺少必要套件，腳本在首次執行時會自動在背景以 pip 完成安裝。

如欲手動安裝依賴，可執行：
```bash
pip install requests python-docx openpyxl python-pptx
```

## 🚀 使用方式與指令範例

當您在與支援本技能的 AI Agent 對話時，只需在對話中 **@ 標註** 檔案，並下達簡繁轉換的指令，Agent 將會為您列出檔案清單確認，並在背景調用核心腳本：

```powershell
python "scripts/convert.py" "<檔案或資料夾路徑>" --mode "<轉換模式>"
```

### 🗣️ 對話指令範例：
* **簡轉繁並台灣化**：
  * *「請幫我把 @會議簡報.pptx 進行簡轉繁，並轉換為台灣用語。」*
  * *「把整個 @文件資料夾 台灣化。」*
* **繁轉簡並中國化**：
  * *「請幫我把 @合約書.docx 繁轉簡，並使用大陸的習慣用語。」*
